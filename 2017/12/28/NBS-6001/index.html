登录主机

NBS-6001 设备出厂默认网络设置为 192.168.1.188，首次使用时请注意，将主机接入 192.168.1.0 网段的网络中。

待主机上电后，稍待短暂的预启动时间，待主机 Run 灯闪烁后，直接在电脑端浏览器中输入 192.168.1.188/pages/login.html，就可以的登录到后台配置页面

默认的登录名为 newbest 密码 pass

注意：主机一旦通过App扫码绑定，则该用户名失效；需使用已绑定的用户身份登陆

配置系统

成功登陆主机后，可以进入系统后台开始系统配置过程。

在这里，工程师将可以完成对系统项目的：

  • 区域布局及设备添加
  • 智慧居家场景配置
  • 智能化规则管理
  • 主机参数设置
  • 系统操作

在参数设置页面中，提供了以下五个设置功能：

IP地址修改 配置主机的网络参数

主机名称

NewBest 支持用户绑定管理多台智能家居系统，用户在 App 端进行绑定操作后，可以在登录个人账号后，通过主机选择页面切换选择不同的主机。主机选择页面中显示了在该地方所设定的主机名称。

城市选择

用户登录 NewBest App 后，在网络连接正常的情况下，可以在首页查看家庭所在城市的天气情况，工程师需要在本页面配置正确的城市信息，精确到地级市。

添加规则

完成了以上区域、设备及场景的配置后，App 端会生成系统完整的控制界面，但是，智能化系统设计者可能还会提出一些 特殊需求，例如：

  • 每周一到周五，当我回家打开房门时，自动开启热水器
  • 每天早上 7：00，播放舒缓的背景音乐
  • KNX 面板联动空调、电视等非 KNX 系统设备
  • 当出现漏水、漏气等情况时，通过 App 发送报警弹窗
  • 当室内温度高于 30℃ 时，自动开启空调设备
  • 根据室内 PM2.5 值，自动调控新风系统

诸如以上此类“逻辑条例”，必须借助智能化规则来实现。

若把区域、设备乃至场景比作积木，那么智能化规则就是将积木搭成宏伟作品的关键。好的智能化项目必需具备创新、适用的智能化规则。

注意：由同一端口做触发条件的逻辑规则，数量上限为 128 条。

系统管理

恭喜你，已经基本掌握了系统的配置搭建，在开始真正的项目之前，工程师还必须掌握如何对工程项目的数据进行备份、恢复、重置及更新。这些操作均可以通过系统管理页面实现。

此页面分为三部分：

  • 数据管理，提供项目配置数据的备份、恢复等操作
  • 系统升级，提供系统远程升级操作
  • 二维码，提供本机设备信息

数据管理

系统配置完成后，相关配置信息只是缓存在系统运行脚本中，请务必执行保存操作，使配置永久生效

点击页面右上角 保存配置 或数据管理页面下的 主机保存 按钮，主机将会自动重启，同时相关的配置信息（区域、设备、场景及规则）会以文件形式保存于设备中。

系统会将 区域 设备 场景 以 homeConfig.json 文件名保存，将 规则 以 ruleConfig.json 保存。通过点击 配置下载 和 规则下载 按钮，可以分别下载此两份配置文件到工程师电脑上。

此后，通过 配置上传 功能，可以随时恢复或是用于批量项目的快速复制。

分别上传已备份的 homeConfig.json 与 ruleConfig.json 文件后，执行配置重置操作即可。

系统还为集团性客户提供了 App 界面自定义的功能，

通过 图片上传 功能，可以修改 App 中标题栏 Logo 及首页背景图，请注意相关文字提示，上传对应大小的图片可以保证最优的效果。

上传后，通过手机登录主机即可查看，登录后，App 会将此两幅图片保存在手机内存中。
如若更换图片重新上传，需要在手机 App 端清除已有缓存记录，清除缓存按钮隐藏于登录页面底部 NewBest 处。

系统升级

NewBest 团队始终致力于为智能家居行业客户带来优质的解决方案和更加智能的系统功能体验。随着新技术爆炸及新应用的出现，NewBest 团队必然会在 NBS-6001 硬件基础上，不断迭代完善其软件系统功能，而这些软件系统功能的升级，我们不会收取额外的费用。

我们会将同一硬件设备的软件升级包发布在 NewBest 企业云上，所有 NewBest 集成商客户，只需将对应的硬件设备接入 Internet 网络，通过系统升级页面选择升级即可。

升级时，注意与当前版本进行比对，选择新版本进行操作。

此外，我们还提供了局域网升级的方式，可以由工程师事先下载准备好升级包，由局域网内上传主机进行升级操作。

请注意，升级后，需要手动重启才可生效。

二维码

所有 NBS-6001 设备均具有唯一的 ID 编号，此编号用于用户 App 端绑定与远程控制。

建议集成商在项目实施时，将此编号打印并放置在明显位置，以便最终用户的使用。

NewBest 提供给用户远程控制附加功能，用户使用时首先需要在 App 端注册个人账号并登陆，之后，点击添加主机按钮-自动搜索，App 将自动搜索局域网内主机设备，选择需要绑定的主机点击绑定即可：

管理员绑定完成后，后续添加绑定的成员只需提交申请留言，管理员就会受到相应申请，进行相关授权即可：

绑定完成后，管理员可进入设置页面的 成员管理 对已绑定的成员进行权限修改或踢出：
