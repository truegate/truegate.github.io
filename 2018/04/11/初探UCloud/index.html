初探UCloud

发表于 2018-04-11 | 分类于 技术 |

计算

云主机 UHost

UHost 云计算实例，是 UCloud 最为核心的服务。 有些服务，如 IP、镜像、云硬盘等必须与云主机结合后使用，其他服务，如数据库、缓存、对象存储等可以和云主机结合共同构建 IT 架构。

私有专区 UDHost

私有专区云主机和其他用户云主机在资源层物理隔离，以满足合规与独享资源池的需求。私有专区主机和普通云主机虽然物理隔离，但网络互通，操作完全一致。

物理云主机 UPHost

提供专用物理服务器，没有经过虚拟化，提供了更卓越的计算性能，满足核心应用场景对高性能及稳定性的需求，同时可以和其他云产品灵活结合使用。

突出特点

  • 满足特殊需求：专门为特殊应用场景打造的专属服务器配置，资源优质、价格合理。
  • 性能卓越：采用高规格硬件与I/O设备的专属物理云主机，提供卓越的服务器性能，满足您高性能应用场景的需求。
  • 不支持虚拟化：物理云主机使用固定的Mac，因此不支持虚拟化。也避免了虚拟化系统导致的性能消耗。

混合云 UHybrid

通过公有云＋私有部署＋专线网络为用户提供行业解决方案，重点解决存量IT资源合理利用；金融等行业安全合规性；企业上云保持业务稳定，架构平滑过渡；多数据中心容灾等场景和需求，最终帮助用户更快更简单的使用云计算。

容器集群 UDocker

类 Kubernetes 容器服务平台。

典型应用场景

  • VM 计算场景
  • 微服务

通用计算 UGC

适用于大规模并行计算的分布式云服务。通用计算可支持海量计算任务并发规模，系统自动完成资源及调度，并按实际使用量计费。

UGC 是基于 Serverless 的大规模分布式并行计算服务，您只需要将集成了业务代码的 Docker 镜像上传至 UGC 镜像仓库，便可通过 API 多次提交针对该镜像的计算任务 ，UGC 负责调度和管理用来运行您业务的服务器集群，拉起 Docker 镜像以运行您的代码。

典型应用场景

  • 数据分析
  • 视频转码

自动伸缩 UAS

自动伸缩是一个自动管理服务虚拟机集群的产品

容器服务 CaaS

CaaS 是基于融合容器技术打造的容器服务平台，可以为应用中心方便地部署持久化运行的容器应用，灵活配置容器实例编组，弹性使用物理资源。

容器镜像库UHub

UHub可使开发人员轻松存储、管理和部署 Docker 容器镜像。UHub 与 UDocker 集成，大大简化了生产工作流程的开发。

突出特点

  • 免费：每个项目可以创建一个免费的镜像仓库。

计算工厂 UCF

UCF 是一种提供大规模计算资源、海量存储资源的分布式离线计算平台服务。

突出特点

  • 海量并发计算能力：支持十万级别核并发规模，快速完成计算和数据的调度。
  • 管理灵活方便：提供管理机器和外网IP，用户可以登录到计算节点，管理任务。虚拟机实例不带外网，不能绑定EIP。跳板机有外网IP，可随时通过跳板机管理内部虚拟机实例。不需要 vpn 拨入，不需要vnc。

典型应用场景

  • 电影动画渲染
  • 多媒体转码
  • 生物数据分析
  • 金融数据分析计算

网络

基础网络 UNet

是UCloud的网络服务，实现对网络资源的管理。包含的子产品包括：外网弹性IP，外网防火墙，内网虚拟IP，带宽包，共享带宽，域名要素引擎及跨域管理通道。

私有网络 VPC

虚拟私有网络 VPC 提供用户安全、隔离、IP地址可配置的网络环境。

突出特点

  • NAT 网关：可以实现网络中未绑定 EIP 的主机访问外网。

内网域名解析 UDEE

基于UCloud云平台的内网DNS解析系统，能够为云平台中的资源提供基于内网传输的自定义域名解析功能。

VPN 网关 IPSec VPN

VPN网关服务，提供可容灾的高可用VPN服务，需要配合用户在UCloud的VPC、用户的本地网关及公网服务三者共同使用。 用户可选用多种加密及认证算法，保证隧道的可靠性。

全球动态加速 PathX

各地区用户可实现就近接入，并通过PathX将请求转发回源站，有效规避跨国网络拥塞导致的响应慢、丢包等问题。全球多个地域分布UCloud网络节点形成加速环路，各个节点之间走专线。

跨域通道 UDPN

UDPN 为 UCloud 云平台中跨地域的各个数据中心间，提供低延迟、高质量的跨地区内网数据传输的能力。

负载均衡 ULB

通过在多台云主机间实现应用程序流量的自动分配，实现故障自动切换，提高业务可用性，并提高资源利用率。

存储和分发

文件存储 UFS

UFS 面向 UCloud 云主机提供简单、可扩展、高可靠、高性能的标准 POSIX 文件共享访问方式。在实现存储空间的在线灵活扩展的同时，也在多主机实例之间实现文件空间的共享，极大提升了存储空间的使用效率，降低存储成本。

突出特点

  • 跨可用区共享挂载
  • 容量与弹性
  • 安全可靠：白名单机制

典型应用场景

  • 负载均衡服务的数据共享
  • 日志汇总存储及分析
  • 企业个人文件存储

云硬盘 UDisk

作为云计算场景基础块存储产品为云主机提供持久化存储空间的块设备硬盘。其具有独立的生命周期，基于网络分布式访问，为云主机提供的数据大容量、高可靠、可扩展、高易用、低成本的硬盘。

典型应用场景

  • 小型关系数据库场景与对数据可靠性有一定要求的开发测试场景
  • 对于容量有要求的大文件顺序读写场景（如：hadoop离线数据分析场景）
  • 备份场景

开放式分发节点 UODN

UODN 在全国的上百个数据中心，为用户提供开放式的边缘计算、存储、网络资源。产品采用分布式Docker容器管理技术，让客户可以通过软件自定义内容分发服务。

突出特点

典型应用场景

  • 视频直播
  • 监控平台
  • 网络爬虫
  • AI-Service

开放式分发节点 UODN2.0

UODN2.0 产品为用户在 UCloud 遍布全国的数据中心内提供计算、存储、网络等资源，节点以 VM 形式提供，“相当于”为用户在各地机房内提供“UCloud 云主机”产品。

典型应用场景

  • 视频直播
  • 监控平台
  • 网络爬虫
  • AI-Service

对象存储 UFile

UFile 是为互联网应用提供非结构化文件存储的服务；相对于传统硬盘存储，对象存储具有存储无上限、支持高并发访问、成本更低等优势。

典型应用场景

  • 图片处理服务

云分发 UCDN

UCDN 将用户的加速内容分发至部署在全球的近500个服务节点，并进行智能调控和缓存，为用户计算最近的访问节点，提供给客户更好更快的网络体验。

归档存储 UArchive

UArchive 是 UCloud 提供的一种低成本、高可靠的数据归档服务，适合于海量数据的长期归档、备份。用户可以将数据进行几个月、几年甚至几十年的长时间存储。

突出特点

  • 成本低廉
  • 数据持久性
  • 数据安全性
  • 存储扩展性

    典型应用场景

  • 低成本备份
  • 数据归档
  • 取代磁带库

消息队列 UMQ

UMQ 是广泛应用于企业互联网架构核心中的分布式消息服务，助您实现分布式架构中服务间的异步解耦，提高服务整体的可靠性与扩展性。

突出特点

  • 目前 UMQ 只支持至少一次，订阅端自行实现去重
  • 单条消息最大 64KB

数据方舟 UDataArk

UDataArk 是为UCloud云主机磁盘提供连续数据保护的服务。支持在线实时备份、具有精确到秒级的数据恢复能力。

突出特点

  • 在线备份，无需暂停业务：备份快照时无需暂停业务或停止磁盘读写，不影响线上业务，不损耗磁盘I/O性能。
  • 精确到秒级恢复，实时数据保护：支持12小时内恢复任一秒，24小时内任意整点时刻数据恢复，最大限度避免数据丢失。

数据库

云数据库 UDB

UDB 基于成熟云计算技术的高可用、高性能的数据库服务，完全兼容 MySQL(5.1/5.5/5.6)、MongoDB(2.4/2.6/3.0)、PostgreSQL(9.4)、SQL Server，同时提供备份、监控等全套解决方案，彻底解决数据库运维的烦恼。

云内存存储 UMem

UMem 是基于内存的缓存服务，支持海量小数据的高速访问。UMem 可以极大缓解后端存储的压力，提高网站或应用的响应速度。支持 Key-Value 的数据结构，兼容 Memcached 协议的客户端都可与 UMem 进行通信。

典型应用场景

  • 访问频度极高的业务
  • 大型促销类业务
  • 带有计数器的库存系统

数据传输 UDTS

UDTS 是一种支持数据源之间数据交互的数据服务。目前提供了数据迁移服务，支持关系型数据库之间的数据迁移。

典型应用场景

  • 云主机 UHost上 自建的 MySQL 数据库迁移至云数据库 UDB 服务
  • 具有公网 IP/Port 的本地 IDC 或其他云上 MySQL 数据库迁移至云数据库 UDB 服务
  • UDB 实例之间的迁移

分布式数据库 UDDB

UDDB 是一种稳定、可靠、容量和服务能力可弹性伸缩的分布式关系型数据库服务。UDDB 高度兼容 MYSQL 协议和语法，支持自动化水平拆分，在线平滑扩缩容，服务能力线性扩展，透明读写分离，具备数据库全生命周期运维管控能力。

突出特点

  • 无线扩容
  • 弹性扩展
  • 快速部署
  • 低成本

    典型应用场景

  • 大型应用
  • 物联网数据
  • 文件索引
  • 大数据存储

突出特点

典型应用场景

容量型KV存储 UKV

UKV 针对有大容量 KV 存储需求的客户，使用固态硬盘存储替代成本非常高的内存存储，在适当降低 QPS 的情况下，能够使得存储的总容量达到 PB 级别。支持常用的 Redis 协议，支持多备份存储来保证数据可靠性，使用 Raft 协议来保证多备份间的数据一致性。

数据库专区 UDBC

UDBC 是用户独享的物理宿主机资源，可选多种配置。购买数据库专区后，用户可以在其上灵活创建多种自定义规格的云数据库。

数据分析

托管集群 UHadoop

UHadoop 是基于 Hadoop 开源框架的大数据处理服务，底层节点独占高性能 SAS 磁盘，有效提升数据处理效率。并提供快速扩展、监控和告警等功能，可更便捷管理集群，从而轻松快速并经济地处理大量的数据。

突出特点

  • HDFS：大数据（TB 以上）文件分块存储、读取，廉价的硬件，块文件备份
  • MapReduce：大数据分析化整为零

    典型应用场景

  • 海量数据
  • 秒级营销

弹性计算节点 UHExt

UHExt(UHadoop Extention) 作为UHadoop产品的补充，专门为用户自建Hadoop集群提供动态伸缩的计算节点服务，您只需提供一个镜像，即可快速、经济、高效的对自建集群计算资源进行弹性扩容，大量节省资源申请和集群部署时间。

分布式消息系统 UKafka

UKafka 是 UCloud 平台中的一款专门处理流式数据的分布式消息产品。通过以创建集群的方式创建 UKafka，能够快速实现 Kafka 以及所依赖的服务的部署。为用户提供快速创建、便于管理、并可弹性伸缩的流式数据处理系统。

云数据仓库 UDW

UDW 是大规模并行处理数据仓库产品，提供 Greenplum 和 Udpg 两种可选的类型。Greenplum 是 EMC 开源的数据仓库，Udpg 是基于PostgreSQL 开发的大规模并行、完全托管的 PB 级数据仓库服务。UDW 支持 JSON 类型，可用通过 SQL 让数据分析更简单、高效，为互联网、物联网、金融、电信等行业提供丰富的业务分析能力

典型应用场景

  • 基于 UDW 实现网络流分析

ES服务 Elasticsearch

UES 基于 Elasticsearch 构建，是一个分布式多用户能力的全文搜索引擎服务。UES通过创建集群的方式来提供服务，能够快速实现集群的部署，为客户提供对存储的海量数据进行实时检索和分析的能力。

突出特点

典型应用场景

  • 日志管理分析
  • 全文搜索
  • 文档存储
  • 数据存储（数据库功能）
  • 传统数据库数据分析

调度系统 UScheduler

UScheduler 是基于大数据分析平台的任务调度服务，可无缝接入UHadoop。目前支持 MapReduce、Hive SQL、Spark、SparkSQL 任务，并支持按日期选择方式和 Crontab 方式设置调度时间。任务上支持复杂的任务逻辑编排、失败重启、超时告警、失败告警等。

典型应用场景

  • 任务流管理
  • 工作流可视化
  • 任务日志查看

人工智能

AI实验室 Vitality

AI实验室目的是提供从数据清洗,模型训练,模型验证，模型应用的一整套解决方案，并且提供了在线代码编辑功能。目前产品只支持TensorFlow模型训练，后续我们会不断完善其他模块，并且提供除了TensorFlow之外的其他深度学习框架。

AI在线服务 UAI-Inference

UAI-Inference （UCloud AI online Inference）是面向AI在线inference服务的大规模分布式计算平台。可提供数万的AI在线服务节点，系统自动完成AI请求的负载均衡，并自动实行节点动态扩容和回收并按实际使用量计费。

典型应用场景

  • 图像识别在线服务
  • 自然语言处理在线服务

AI训练服务 UAI-Train

UAI-Train(UCloud AI Train) 是面向 AI 训练任务的大规模分布式计算平台，基于 UCloud 性能强大的 GPU 云主机集群构建，可同时提供多种配置的 GPU 训练节点，包括最大支持 1 机 4 卡 P40 节点提供总共高达 48TFlops 的单精度计算能力，为 AI 训练任务提供充足的计算能力。

UAI-Train平台提供一站式训练任务托管服务，客户无需担心AI训练过程中GPU节点分配、数据上传、下载、节点容灾等问题。UAI-Train平台采取按需收费模式，根据训练任务实际消耗资源收费，收费更灵活、便捷。UAI-Train平台同时提供训练日志转发和TensorBoard转发功能，让客户可以实时监控训练进度。

UAI-Train平台可以满足图像识别、自然语言处理等多个AI领域的算法训练任务。

典型应用场景

  • 图像识别算法训练任务
  • 自然语言处理算法训练任务

视频服务

云点播 UVideo

UVideo 为开发者和企业用户提供包括音视频存储、转码、截图、水印、CDN 加速、多终端播放器 SDK 工具包（Web/Android/iOS）和 API 等服务。

突出特点

  • 极速视频转码：强大的分布式转码集群，极速的转码体验，支持主流视频格式转码，支持多码流多格式输出。
  • CDN 全网加速：多年 CDN技 术积累，稳定快速，多线 BGP 接入，覆盖全网，企业级安全防护，为用户提供优质服务

    典型应用场景

  • 大规模分布式转码服务
  • 提升不同终端的播放体验

云直播 ULive

ULive 提供高清流畅、低延迟、高并发的整套解决方案。包括实时转码，切片存储，分发加速，内容保护等核心功能。带给终端用户流畅的访问体验，简化相关部署运维工作，帮助视频直播业务快速上线。

突出特点

  • 多传输协议：支持 RTMP 及 HLS 协议，RTMP 适用于低延迟强互动性场景，HLS 适用于高并发低互动性场景
  • 直播实时转码：提供视频、音频转码功能，满足不同客户对转码场景的各类需求
  • 录制转点播：支持直播录制转成 mp4 与 HLS 格式点播文件，自动修复中断的视频内容并提供灵活的文件保存策略。
  • 访问鉴权：支持播放url加密方式的鉴权功能，验证用户访问权限，降低内容盗用风险及成本损失。
  • 推流认证：提供源内容端推流鉴权机制，避免未授权内容推流播放端，避免资源盗用产生的成本损失。
  • 开发接口：支持API接口访问服务端并提供SDK开发包，帮助无缝对接原有业务、降低改造成本、缩短上线周期。
  • 播放端 SDK：提供基于Android/iOS平台的SDK开发包，降低移动端应用程序开发成本、缩短业务上线周期。

    典型应用场景

  • 社交直播
  • 媒体直播
  • 秀场直播
  • 游戏直播

媒体工厂 UMeida

UMedia 为用户提供丰富的多媒体处理功能，包括：视频转码、水印、截图、切片以及鉴黄等，将多媒体数据处理为适应PC、智能电视和移动端等多种场景的形式。帮助用户提高视频处理效率。

突出特点

  • 极速转码
  • 灵活定制
  • 存储分发完美结合

    典型应用场景

  • 传统音视频网站

监控 UMon

UMON 是专为 UCloud 云平台中全线产品及互联网站点服务提供的监控、告警通知服务。监控系统能够根据云平台中资源上报数据，实时展示资源的各项监控指标，并能够通过监控模板批量为资源设置告警阈值，以便资源出现告警时，及时通知用户跟进处理。同时，监控系统更能够对外网站点及服务的监控，掌握外网服务运行状况，保证站点及应用的良性运行。

突出特点

  • 易用高效
  • 无缝接入
  • 告警灵活

高防服务 UAnti-DDos

UAnti-DDoS 高防为已备案的域名或源站 IP（包括非 UCloud 的弹性外网IP）提供 DDoS 攻击防护。当用户的域名或源站 IP（包括非 UCloud 的弹性外网 IP）在遭受大流量的 DDoS 攻击时，可以通过高防 IP 代理源站 IP 面向用户，隐藏源站 IP，将攻击流量引流到高防 IP，确保源站的稳定正常运行。

突出特点

  • 复杂穿墙攻击防护：支持防护各类 ACK、SYN、连接耗尽、CC 透过防火墙
  • 超大流量防护：最大可支持600G的ddos攻击防护能力。

    典型应用场景

  • 适用客户：所有需要网络安全防护的客户。（设备可以不在UCloud上面）

企业应用防火墙 UWAF

WAF 是部署在 WEB 服务器前，用于保护网站抵御来自外部和内部的攻击。企业应用防火墙可以保护所有需要 WEB 应用防护的客户，不仅限于UCloud 的设备和网络上的网站。使用者可以根据自身网站情况定制防护策略，也可以采用系统提供的默认策略进行防护。

典型应用场景

  • 适合用户：所有需要Web应用防护的客户。（设备可以不在UCloud上面）

堡垒机 UHAS

UHAS（UCloud Host Audit System），可以用于在运维人员访问云主机 UHOST 的过程中，充当跳板，具备完备的审计功能。

数据库审计 UDB-Audit

UDAS(UCloud Database Analysis System）对审计和事务日志进行审查，从而跟踪各种对数据库操作的行为。一般审计主要记录对数据库的操作、对数据库的改变、执行该项目操作的人以及其他的属性。这些数据库被记录到UDAS独立的平台中，并且具备较高的准确性和完整性。针对数据库活动或状态进行取证检查时，审计可以准确的反馈数据库的各种变化，对我们分析数据库的各类正常、异常、违规操作提供证据。

突出特点

系统采用三权分立的模式，包括三个平台，系统管理平台、规则管理平台、审计管理平台，各平台的功能与职责不同，权限不同，相互监督。

典型应用场景

证书管理 USSL

USSL（UCloud SSL）SSL证书管理服务为客户提供一站式的证书购买、证书审批、证书上传/下载、证书吊销和证书删除的全生命周期管理。

等保测评 UDBCP

等保测评服务为客户提供一站式的等保解决方案，在控制台提交等保评测申请后，我们会为您联系合适的评测机构。

此外UCloud为客户提供一套针对等保三级（三级以下同样适用）的安全合规方案，为用户提升等保测评得分。

客户进行等保测评时，涉及到UCloud云平台的部分，UCloud提供自身云平台的备案证明、云平台通过测评的证明材料。
